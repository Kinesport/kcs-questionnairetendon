Classification de Mueller-Wohlfahrt
Référence Kinésport — Cliquez pour sélectionner le grade
Classification Mueller-Wohlfahrt (adaptée Kinésport)Référence
Cliquez sur le grade correspondant à la lésion. Cette classification est la référence utilisée dans les formations Kinésport.
Type 1a
Fatigue musculaire fonctionnelle
Surmenage sans lésion structurelle. Douleurs diffuses, pas de creux, IRM normale.
Type 1b
Courbatures sévères (DOMS)
Micro-lésions diffuses post-effort. Douleurs retardées 24-48h, diffuses, sans déficit fonctionnel.
Type 2a
Trouble neuromyofascial
Contracture localisée, tension anormale, point trigger. IRM normale ou hypersignal discret.
Type 2b
Lésion myofasciale (claquage 1)
Déchirure du fascia musculaire. Douleur aiguë localisée, légère ecchymose. IRM : signal fascial.
Type 3a
Déchirure musculaire partielle mineure
Rupture partielle < 50% de la section. Creux palpable possible. IRM : hypersignal < 50%.
Type 3b
Déchirure musculaire partielle majeure
Rupture partielle > 50% de la section. Creux important. IRM : hypersignal > 50%.
Type 4
Rupture totale / avulsion
Rupture complète avec ou sans avulsion. Creux total, hématome important. Chirurgie à discuter.
Contusion
Traumatisme direct
Choc direct sans rupture de fibre. Hématome intramusculaire. Risque de myosite ossifiante.
